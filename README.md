
# 概览
UCloud混合云多云运营管理平台（以下简称：UCMP）是UCloud基于自身多年云服务管理经验推出的一款纯自研私有化软件产品。该产品基于多云、混合云的云计算市场发展趋势，以客户管云痛点为核心，为数据化转型过程中的企业提供便捷、高效的云管理工具。
